# fresholm Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Python package (`fresholm`) that replaces the abandoned `python-olm` dependency in the mautrix Matrix bridge ecosystem with Rust-based vodozemac 0.9 bindings via PyO3 0.28.

**Architecture:** Three-layer stack: (1) Rust/PyO3 bindings (`fresholm._native`) wrapping vodozemac 0.9's Olm and Megolm implementations, (2) Pure Python compatibility layer (`fresholm.compat.olm`) that translates vodozemac's API to match python-olm's exact interface (properties, method names, exceptions, serialization format), (3) Import hook (`fresholm.import_hook`) that patches `sys.modules['olm']` so mautrix bridges work with zero code changes.

**Tech Stack:** Rust 1.94+, vodozemac 0.9, PyO3 0.28, maturin 1.8+, Python 3.10+, pytest, asyncio

## Critical Differences from Original Spec

Research revealed significant API differences between the spec's assumptions and actual library versions:

| Area | Spec Assumed | Actual |
|------|-------------|--------|
| vodozemac | 0.8 | **0.9.0** — `create_outbound_session` takes `SessionConfig`; serialization is two-step (`obj.pickle()` then `pickle_obj.encrypt(&[u8;32])`); `OlmMessage` is an enum |
| PyO3 | 0.22 | **0.28.3** — Bound API mandatory; declarative `#[pymodule]`; `IntoPyObject` trait; `.cast()` replaces `.downcast()` |
| python-olm API | Methods | **Properties** — `identity_keys`, `one_time_keys`, `max_one_time_keys`, `id`, `session_key`, `first_known_index` are all properties in python-olm |
| python-olm methods | `create_outbound_session` | **`new_outbound_session`**, `new_inbound_session`, `remove_one_time_keys(session)` |
| python-olm serialization | bytes key | **String passphrase** — `.pickle("passphrase")` |
| python-olm exceptions | `OlmError` | **`OlmSessionError`**, `OlmGroupSessionError` |
| Subclassing | Not considered | **Required** — mautrix subclasses all olm classes |

```rust
use pyo3::prelude::*;
use pyo3::types::PyDict;
use vodozemac::olm::Account as VzAccount;
use vodozemac::olm::SessionConfig;
use vodozemac::Curve25519PublicKey;

/// Helper: convert a passphrase (arbitrary bytes) to a 32-byte key.
/// Zero-pads if shorter, truncates if longer.
pub(crate) fn passphrase_to_key(passphrase: &[u8]) -> [u8; 32] {
    let mut key = [0u8; 32];
    let len = passphrase.len().min(32);
    key[..len].copy_from_slice(&passphrase[..len]);
    key
}

#[pyclass]
pub struct Account {
    pub(crate) inner: VzAccount,
}

#[pymethods]
impl Account {
    #[new]
    fn new() -> Self {
        Self {
            inner: VzAccount::new(),
        }
    }

    /// Returns {"ed25519": "...", "curve25519": "..."}.
    fn identity_keys<'py>(&self, py: Python<'py>) -> PyResult<Bound<'py, PyDict>> {
        let keys = self.inner.identity_keys();
        let dict = PyDict::new(py);
        dict.set_item("ed25519", keys.ed25519.to_base64())?;
        dict.set_item("curve25519", keys.curve25519.to_base64())?;
        Ok(dict)
    }

    /// Sign a message, returning the base64 signature.
    fn sign(&self, message: &[u8]) -> String {
        self.inner.sign(message).to_base64()
    }

    fn generate_one_time_keys(&mut self, count: usize) {
        self.inner.generate_one_time_keys(count);
    }

    /// Returns {key_id_b64: key_b64, ...}.
    fn one_time_keys<'py>(&self, py: Python<'py>) -> PyResult<Bound<'py, PyDict>> {
        let keys = self.inner.one_time_keys();
        let dict = PyDict::new(py);
        for (id, key) in keys {
            dict.set_item(id.to_base64(), key.to_base64())?;
        }
        Ok(dict)
    }

    fn mark_keys_as_published(&mut self) {
        self.inner.mark_keys_as_published();
    }

    fn max_number_of_one_time_keys(&self) -> usize {
        self.inner.max_number_of_one_time_keys()
    }

    /// Create an outbound Olm session.
    fn create_outbound_session(
        &self,
        their_identity_key: &str,
        their_one_time_key: &str,
    ) -> PyResult<crate::session::Session> {
        let identity = Curve25519PublicKey::from_base64(their_identity_key)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        let otk = Curve25519PublicKey::from_base64(their_one_time_key)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        let session = self
            .inner
            .create_outbound_session(SessionConfig::version_2(), identity, otk);
        Ok(crate::session::Session::from_vz(session))
    }

    /// Create an inbound Olm session from a pre-key message.
    /// Returns (Session, plaintext_bytes).
    fn create_inbound_session(
        &mut self,
        their_identity_key: &str,
        pre_key_message_bytes: &[u8],
    ) -> PyResult<(crate::session::Session, Vec<u8>)> {
        let identity = Curve25519PublicKey::from_base64(their_identity_key)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        let pre_key_msg =
            vodozemac::olm::PreKeyMessage::from_bytes(pre_key_message_bytes)
                .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        let result = self
            .inner
            .create_inbound_session(identity, &pre_key_msg)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        Ok((
            crate::session::Session::from_vz(result.session),
            result.plaintext,
        ))
    }

    /// Serialize the account with a passphrase. Returns encrypted string.
    fn pickle(&self, passphrase: &[u8]) -> PyResult<String> {
        let key = passphrase_to_key(passphrase);
        let serialized = self.inner.pickle();
        Ok(serialized.encrypt(&key))
    }

    /// Restore an Account from an encrypted string.
    #[staticmethod]
    fn from_pickle(encrypted: &str, passphrase: &[u8]) -> PyResult<Self> {
        let key = passphrase_to_key(passphrase);
        let deserialized = vodozemac::olm::AccountPickle::from_encrypted(encrypted, &key)
            .map_err(|e| crate::errors::OlmAccountError::new_err(format!("{e}")))?;
        Ok(Self {
            inner: VzAccount::from_pickle(deserialized),
        })
    }

    fn __repr__(&self) -> String {
        let ed = self.inner.identity_keys().ed25519.to_base64();
        format!("<Account ed25519={}>", &ed[..8])
    }
}
```

> **Note:** The `create_inbound_session` uses `PreKeyMessage::from_bytes()`. If vodozemac 0.9 doesn't expose this directly, use `OlmMessage::from_bytes()` and match the `PreKey` variant. Adjust during compilation.

```rust
use pyo3::prelude::*;
use vodozemac::olm::{OlmMessage, Session as VzSession};

use crate::account::passphrase_to_key;

#[pyclass]
pub struct Session {
    pub(crate) inner: VzSession,
}

impl Session {
    pub fn from_vz(inner: VzSession) -> Self {
        Self { inner }
    }
}

/// Result of Session.encrypt(). Holds the message type and ciphertext bytes.
#[pyclass]
#[derive(Clone)]
pub struct EncryptedMessage {
    pub(crate) raw: Vec<u8>,
    pub(crate) msg_type: u32,
}

#[pymethods]
impl EncryptedMessage {
    /// 0 = PreKey, 1 = Normal.
    #[getter]
    fn message_type(&self) -> u32 {
        self.msg_type
    }

    /// Raw ciphertext bytes (wire format).
    #[getter]
    fn ciphertext(&self) -> Vec<u8> {
        self.raw.clone()
    }

    fn __repr__(&self) -> String {
        let label = if self.msg_type == 0 { "PreKey" } else { "Normal" };
        format!("<EncryptedMessage type={label}>")
    }
}

#[pymethods]
impl Session {
    /// Encrypt plaintext. Returns an EncryptedMessage with message_type and ciphertext.
    fn encrypt(&mut self, plaintext: &[u8]) -> EncryptedMessage {
        let msg = self.inner.encrypt(plaintext);
        let (msg_type, raw) = match &msg {
            OlmMessage::PreKey(_) => (0u32, msg.to_bytes()),
            OlmMessage::Normal(_) => (1u32, msg.to_bytes()),
        };
        EncryptedMessage { raw, msg_type }
    }

    /// Decrypt a message. Takes message_type (0=PreKey, 1=Normal) and ciphertext bytes.
    /// Returns plaintext bytes.
    fn decrypt(&mut self, message_type: u32, ciphertext: &[u8]) -> PyResult<Vec<u8>> {
        let msg = OlmMessage::from_bytes(ciphertext)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        // Verify the message type matches what was passed
        let type_ok = match (&msg, message_type) {
            (OlmMessage::PreKey(_), 0) | (OlmMessage::Normal(_), 1) => true,
            _ => false,
        };
        if !type_ok {
            return Err(crate::errors::OlmSessionError::new_err(
                "message_type does not match ciphertext content",
            ));
        }
        self.inner
            .decrypt(&msg)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))
    }

    fn session_id(&self) -> String {
        self.inner.session_id()
    }

    fn has_received_message(&self) -> bool {
        self.inner.has_received_message()
    }

    fn pickle(&self, passphrase: &[u8]) -> PyResult<String> {
        let key = passphrase_to_key(passphrase);
        let serialized = self.inner.pickle();
        Ok(serialized.encrypt(&key))
    }

    #[staticmethod]
    fn from_pickle(encrypted: &str, passphrase: &[u8]) -> PyResult<Self> {
        let key = passphrase_to_key(passphrase);
        let deserialized = vodozemac::olm::SessionPickle::from_encrypted(encrypted, &key)
            .map_err(|e| crate::errors::OlmSessionError::new_err(format!("{e}")))?;
        Ok(Self {
            inner: VzSession::from_pickle(deserialized),
        })
    }

    fn __repr__(&self) -> String {
        let id = self.inner.session_id();
        let short = if id.len() > 8 { &id[..8] } else { &id };
        format!("<Session id={short}>")
    }
}
```

> **Note:** This implementation uses `OlmMessage::to_bytes()` and `OlmMessage::from_bytes()`. If these methods don't exist in vodozemac 0.9, alternatives: (a) use serde serialization, (b) match on variants and access inner `Message`/`PreKeyMessage` bytes individually. Adjust during compilation.

            .map_err(|e| crate::errors::OlmGroupSessionError::new_err(format!("{e}")))?;
        let result = self
            .inner
            .decrypt(&msg)
            .map_err(|e| crate::errors::OlmGroupSessionError::new_err(format!("{e}")))?;

    """Error in Olm account operations."""

    pass


def _passphrase_to_bytes(passphrase) -> bytes:
    """Convert a passphrase to bytes.

    python-olm accepts str or bytes. We normalize to bytes for the Rust layer.

    if isinstance(passphrase, str):

from fresholm._native import (

    InboundGroupSession as _NativeInboundGroupSession,

)

from .types import (
    OlmError,
    OlmSessionError,
    OlmGroupSessionError,
    OlmAccountError,
    CryptoStoreError,
    _passphrase_to_bytes,
)


class Account:
    """python-olm compatible Account backed by vodozemac.

    Subclassable. mautrix inherits from this.
    """

    def __init__(self):
        self._native = _NativeAccount()

    @property
    def identity_keys(self):
        """Dict with 'ed25519' and 'curve25519' base64 public keys.

        python-olm exposes this as a property, not a method.
        """
        return self._native.identity_keys()

    @property
    def one_time_keys(self):
        """Dict of {key_id: key_value} for unpublished one-time keys.

        python-olm returns {"curve25519": {id: key, ...}}.
        """
        raw = self._native.one_time_keys()
        return {"curve25519": dict(raw)}

    @property
    def max_one_time_keys(self):
        """Maximum number of one-time keys the account can hold."""
        return self._native.max_number_of_one_time_keys()

    def generate_one_time_keys(self, count):
        """Generate `count` new one-time keys."""
        self._native.generate_one_time_keys(count)

    def mark_keys_as_published(self):
        """Mark current one-time keys as published (clears them from one_time_keys)."""
        self._native.mark_keys_as_published()

    def sign(self, message):
        """Sign a message. Accepts str or bytes. Returns base64 signature string."""
        if isinstance(message, str):
            message = message.encode("utf-8")
        return self._native.sign(message)

    def new_outbound_session(self, identity_key, one_time_key):
        """Create outbound Olm session. Returns a Session object.

        Args:
            identity_key: Recipient's curve25519 identity key (base64 str).
            one_time_key: Recipient's one-time key (base64 str).
        """
        native_session = self._native.create_outbound_session(
            identity_key, one_time_key
        )
        session = Session.__new__(Session)
        session._native = native_session
        session._creation_plaintext = None
        return session

    def new_inbound_session(self, sender_key, message):
        """Create inbound Olm session from a pre-key message.

        Args:
            sender_key: Sender's curve25519 identity key (base64 str).
            message: An OlmPreKeyMessage instance.

        Returns:
            Session object.
        """
        native_session, plaintext = self._native.create_inbound_session(
            sender_key, message.ciphertext
        )
        session = Session.__new__(Session)
        session._native = native_session
        session._creation_plaintext = plaintext
        return session

    def remove_one_time_keys(self, session):
        """No-op. vodozemac removes one-time keys automatically during session creation."""
        pass

    def pickle(self, passphrase):
        """Serialize the account. Returns bytes."""
        key = _passphrase_to_bytes(passphrase)
        return self._native.pickle(key).encode("utf-8")

    @classmethod
    def from_pickle(cls, data, passphrase):
        """Restore an Account from serialized bytes."""
        key = _passphrase_to_bytes(passphrase)
        if isinstance(data, bytes):
            data = data.decode("utf-8")
        instance = cls.__new__(cls)
        instance._native = _NativeAccount.from_pickle(data, key)
        return instance

    Subclassable. mautrix inherits from this.

        self._native = None
        self._creation_plaintext = None

        """Session ID string. python-olm exposes this as a property."""
        return self._native.session_id()

        Args:
            plaintext: str or bytes to encrypt.

        Returns:

        return _wrap_encrypted(result)

"""In-memory crypto store for testing and lightweight deployments."""

from typing import Optional

from .base import BaseCryptoStore


class MemoryCryptoStore(BaseCryptoStore):
    def __init__(self):
        self._account = None
        self._sessions: dict[str, list] = {}
        self._group_sessions: dict[str, object] = {}

    async def put_account(self, account) -> None:
        self._account = account

    async def get_account(self) -> Optional[object]:
        return self._account

    async def put_sessions(self, sender_key: str, sessions: list) -> None:
        self._sessions[sender_key] = list(sessions)

    async def get_sessions(self, sender_key: str) -> list:
        return list(self._sessions.get(sender_key, []))

    async def add_session(self, sender_key: str, session) -> None:
        if sender_key not in self._sessions:
            self._sessions[sender_key] = []
        self._sessions[sender_key].append(session)

    async def update_session(self, sender_key: str, session) -> None:
        sessions = self._sessions.get(sender_key, [])
        for i, s in enumerate(sessions):
            if s.session_id() == session.session_id():
                sessions[i] = session
                return
        sessions.append(session)
        self._sessions[sender_key] = sessions

    async def delete_session(self, sender_key: str, session_id: str) -> None:
        sessions = self._sessions.get(sender_key, [])
        self._sessions[sender_key] = [
            s for s in sessions if s.session_id() != session_id
        ]

    async def delete_all_sessions(self, sender_key: str) -> None:
        self._sessions.pop(sender_key, None)

    async def put_group_session(
        self, room_id: str, sender_key: str, session_id: str, session
    ) -> None:
        key = f"{room_id}|{sender_key}|{session_id}"
        self._group_sessions[key] = session

    async def get_group_session(
        self, room_id: str, sender_key: str, session_id: str
    ) -> Optional[object]:
        key = f"{room_id}|{sender_key}|{session_id}"
        return self._group_sessions.get(key)

    async def has_group_session(
        self, room_id: str, sender_key: str, session_id: str
    ) -> bool:
        key = f"{room_id}|{sender_key}|{session_id}"

## Known Limitations & Future Work

These items are **not** in scope for this plan but should be tracked:

1. **`Session.matches()` returns `False`** — needs proper implementation using session key comparison. mautrix will fall through to creating new inbound sessions, which works but is less efficient. Fix by comparing `PreKeyMessage` base key with session keys.

2. **Serialization format incompatibility** — vodozemac native serialized data is NOT compatible with libolm/python-olm serialized data. Existing bridges upgrading from python-olm will need to clear their crypto stores or implement migration using vodozemac's `from_libolm_pickle()` functions.

3. **PkEncryption / PkDecryption** — Not implemented (mautrix doesn't use them). Available in vodozemac behind the `insecure-pk-encryption` feature flag.

4. **PkSigning** — Used by mautrix for cross-signing. Needs implementation wrapping vodozemac's Ed25519 signing.

5. **`_libolm` FFI compatibility** — mautrix falls back to direct `_libolm.ffi`/`_libolm.lib` for `Session.describe()`. Our `describe()` method provides an alternative, but code that directly accesses `_libolm` will fail.

6. **AsyncPG / SQLite crypto stores** — Only MemoryCryptoStore is implemented. Production bridges need persistent stores.

7. **Type stubs** — `.pyi` files for IDE support not yet created.

8. **Windows testing** — CI workflow includes Windows but local testing is Linux-only.
